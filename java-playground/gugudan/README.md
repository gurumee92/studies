* OpenJDK 17
* Git 2.34.1
* IntelliJ 혹은 그에 준하는 IDE

* 사용자 값을 입/출력하는 방법
* 변수 데이터 타입 반복문 조건문 등의 기초 문법
* 콜렉션(배열), 메소드, 클래스를 활용한 리팩토링
